<h1 align="center"> Игральные кости </h1>
<p>Приложение имитирующее бросание кубиков.</p>

<p>По клику на экран показывается случайная картинка кубиков с количеством точек от 1 до 6.</p>

<p>Технологии:
Jetpack Compose</p>
